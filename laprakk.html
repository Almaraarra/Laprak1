Laporan Teori

  1. Pertanyaan seputar tag
    1. Apa yang dimaksud dengan tag pada HTML?
      tag adalah simbol khusus (markup) berupa dua karakter "<" dan ">" yang mengapit suatu teks sebagai nama tag.
      Contohnya tag < body> adalah tag dengan nama body. Sebagian besar tag memiliki tag pembuka dan tag penutup yang membatasi pengaruh dari tag tersebut. Namun, terdapat sebagian kecil tag yang tidak memiliki tag penutup.

      Penutup tag ditambah simbol "/" setelah tanda "<",
      misalnya < /body > yang merupakan penutup tag < body>.

      Berikut adalah karakteristik dari tag pada HTML:
      • Tag boleh ditulis dengan huruf besar (kapital), kecil atau campuran, karena sifatnya yang incase sensitive.
        Misalnya: < BODY >, < body > atau < Body > semuanya memiliki hasil yang sama. Namun, untuk standarnya lebih baik menggunakan huruf kecil semua.
      • Tag bisa berisi tag lain. Tag yang dibuka dahulu ditutup terakhir, sedangkan tag yang dibuka terakhir ditutup pertama. Namun, kesalahan dalam struktur ini tidak mengakibatkan error. Misalnya: <\p>I like < b > HTML < /b >< /p >< /li >


    2. Apa itu pembuka dan penutup tag? Apakah hal tersebut terdapat pada semua jenis tag?
      • Tag Pembuka: merupakan tag yang digunakan untuk memulai elemen.
        Contohnya, < h1> untuk heading atau < p> untuk paragraf.
      • Tag Penutup : adalah tag yang digunakan untuk menandai akhir dari elemen tersebut. Tag penutup ditulis dengan menambahkan garis miring (/) sebelum nama tag.
        Contohnya, < /h1> dan < /p>.

        • Contoh :
        < h1> Judul Utama < /h1>
        < p> Ini adalah paragraf pertama.
        < /p> Di sini sebagai penutup paragraf,
        < h1> adalah tag pembuka dan < /h1> adalah tag penutup untuk judul utama.
        Begitu juga dengan < p> dan < /p> untuk paragraf.

      • Apakah Semua Jenis Tag Memiliki Pembuka dan Penutup?
        Tidak semua jenis tag memiliki pasangan pembuka dan penutup. Sebagian besar tag di HTML memang ditulis berpasangan, tetapi ada juga tag yang berdiri sendiri.
        Misalnya:
        • Tag dengan pasangan: < div>< /div>, < span>< /span>
        • Tag berdiri sendiri: < br>, < img>, < hr>

  2. Pertanyaan seputar element
    1. Apa yang dimaksud dengan element?
      Element adalah bagian dari skrip HTML yang terdiri dari tag pembuka, isi element dan tag penutup. Jika suatu element ditampilkan pada browser, maka yang akan di tampilkan adalah isi elementnya.
      Misalnya: ‹ p> Aku senang belajar HTML< /p>
      Jika ditampilkan pada browser, maka hanya tulisan "Aku senang belajar HTML" yang akan tampil di browser.

    2. Apa yang dimaksud dengan tagname , attribute , dan content pada element?
      • Tagname
        Setiap elemen yang digunakan dalam dokumen HTML diberi nama tagname, yang membantu browser memahami fungsi dan tujuan dari setiap bagian konten.
        Contoh tagname yang umum meliputi:
        • Paragraf: < p>
        • Judul H1-H6: < h1>, < h2>, ..., < h6>< /h6>
        • Link: < a>< /a>
        • Gambar: < img>
        • Blok Tebal/Batang/Kursif/Teks Biasa: < strong>, < em>, < i>,< code>

        Misalkan Anda ingin membuat paragraf dengan kalimat, < p>Saya sedang belajar HTML< /P>, dan tagname "< p>< /P>" menunjukkan bahwa elemen tersebut adalah paragraf.

      • Attribute
        Attribut merupakan elemen tambahan yang di gunakan untuk memberikan informasi tambahan untuk beberapa elemen tertentu. ditulis dalam bentuk "nama-atribut=nilai-atribut" dan berfungsi berdasarkan jenis elemen yang digunakan.
        Beberapa contoh attribut yang umum digunakan adalah:
        • URL Link: href - Digunakan untuk menetapkan alamat website yang terkait dengan link.
          < a href="https://www.example.com">Clik Here< /a>
        • Alt Text for Images: alt – Digunakan untuk memberikan deskripsi text alternative bagi image jika gambar tidak dapat dimuat. < img src="image.jpg" alt="Image Description">
        • Id Unique Identifier: id – Digunakan untuk memberikan identifikasi unik pada satu elemen saja. < div id="main-content">< /div>

      • Content
        Data yang ditampilkan oleh elemen HTML disebut konten. konten bisa berupa teks, gambar, video, formulir input user, atau kombinasi dari semua ini.
        Contoh penggunaan content :
        • Teks biasa dalam paragraph,
        • Deskripsi lengkap tentang produk di toko online,
        • Informasi kontak perusahaan

    3. Apakah element dapat ditumpuk? Apa yang terjadi bila demikian?
      bisa, element bisa di tumpuk dengan element lainnya, dan hal yang akan terjadi jika element di tumpuk adalah element di bawahnya ke tutup dengan element di atasnya.
      Misalnya, di pemrograman web, dengan CSS z-index, elemen dengan nilai z-index lebih tinggi akan tampil di atas elemen lain.

  3. Pertanyaan seputar anchor
    1. Apa yang dimaksud dengan anchor?
      Anchor merupakan element utama yang di gunakan untuk membuat hyperlink,dimana element ini memungkinkan kita untuk mengklick ke halaman lain, bagian tertentu dari halaman yang sama, bahkan melakukan tindakan lain, seperti mengirim email.
      Anchor tag ditulis dengan simbol < a>. Tag ini sendiri tidak terlalu berfungsi jika tidak dilengkapi dengan atribut href. Atribut href menentukan URL tujuan dari tautan.
      < a href="https://www.example.com">Kunjungi Example< /a> Dalam contoh ini, ketika user mengklik "Kunjungi Example," maka user akan diarahkan ke https://www.example.com.

    2. Bagaimana caranya anchor dapat berfungsi, baik dari sisi pendefinisian maupun pengaplikasiannya?
      Pengaplikasian Anchor Tag :
      • Membuat Tautan Antar Halaman
        Anchor tag digunakan untuk membuat tautan antar halaman. Contoh : user ingin membuat tautan ke halaman lain di situs web. < a href="/about-us.html">Tentang Kami< /a>
      • Melompat Ke Bagian Spesifik Halaman
        Untuk melompat ke bagian spesifik dalam halaman yang sama, user dapat menggunakan URL fragment (#ID_Lokasi).
        < !-- Lokasi spesifik dalam halaman -->
        < span id="lokalisasi-spesifik">< /span>
        < !-- Tautan untuk melompat ke lokasi spesifik -->
        < a href="#lokalisasi-spesifik">Lompat ke Lokasi Spesifik< /a>

    Jawaban soal no.4

    A.Text

    • Tag untuk memanipulasi konten teks:
      • < b> atau < strong>: Menebalkan teks.
      • < i> atau < em>: Memiringkan teks.
      • < u>: Menggarisbawahi teks.
      • < small>: Menampilkan teks dalam ukuran kecil.
      • < mark>: Menyoroti teks.
      • < sub>: Menampilkan teks sebagai subskrip.
      • < sup>: Menampilkan teks sebagai superskrip.

    • Tag untuk memanipulasi struktur teks:
      • < p>: Membuat paragraf
      • < h1>, < h2>, < h3>, < h4>, < h5>, < h6>: Membuat heading (judul) dengan ukuran yang bervariasi.
      • < br>: Membuat baris baru.
      • < ul>, < ol>, < li>: Membuat daftar tidak berurutan (unordered) atau berurutan (ordered).

    B.Image

    • Kemungkinan kesalahan yang terjadi pada tag < img>:
      • Sumber (src) gambar salah atau tidak ditemukan (broken link).
      • Ekstensi file gambar salah atau tidak sesuai format (misalnya .jpg tapi file sebenarnya .png).
      • Kesalahan penulisan direktori atau folder tempat file gambar disimpan.
      • Tidak ada alt text yang menjelaskan gambar jika gagal dimuat.

    • Jika ada teks yang menggantikan gambar:
      • Error Load Gambar: gambar tidak bisa di muat karena kesalahan url atau file tidak didukung.
      • Empty Alt Text: Jika atribut alt elemen < img> tidak memiliki deskripsi, teks tidak akan ditampilkan.
      • Conditional Loading: kesalahan dalam pemuatan sehingga mengakibatkan teks fallback muncul dan gambar tidak muncul
      • CSS Styling Error: Untuk memastikan elemen gambar tidak tersembunyi, perhatikan aturan CSS.
      • JavaScript Manipulation: JavaScript dapat mengganti elemen gambar dengan teks baik secara langsung maupun melalui manipulasi DOM. Maka perlu memperhatikan kode java

    C.Audio
    Langkah - langkah memasukan source audio :

    • Gunakan Elemen < audio>
      digunakan untuk menyematkan file audio ke dalam halaman web.
    • Tambahkan Atribut src
      Gunakan atribut src untuk menentukan URL dari file audio yang ingin Anda masukkan.
    • Contoh kode
      < audio controls> < source src="https://www.example.com/audio-file.mp3" type="audio/mpeg">

    D.Video
    Jika atribut autoplay pada tag video tidak berjalan, beberapa hal yang biasanya terjadi adalah:

    • Perubahan Pola Otomatis Browsers
      Banyak browser memiliki pola otomatis yang berbeda-beda untuk mengizinkan atau melarang autoplay video. Beberapa browser hanya mengizinkan autoplay jika volume sudah dinonaktifkan atau jika pengguna telah berinteraksi dengan halaman web sebelumnya.137
    • Volume Dinonaktifkan
      Menambahkan fitur muted ke autoplay adalah salah satu cara yang umum untuk membuat autoplay berfungsi. Ini akan memulai video secara otomatis tanpa bunyi, menjadikannya lebih mudah digunakan dan diterima oleh banyak browser.126.
    • iOS Safari
      Jika fitur playsinline ditambahkan ke Safari iOS, autoplay tidak akan berfungsi. Ini disebabkan oleh peraturan ketat yang dibuat oleh iOS untuk mencegah penggunaan data yang tidak diinginkan dan gangguan.126.
    • Fallback Kontrol
      Jika autoplay tetap tidak berfungsi, kita dapat menambahkan kontrol manual seperti tombol play/pause untuk memberi pengguna kemampuan untuk menyesuaikan pemutaran video dengan lebih baik.126.
